# Kuba's Web Dev Hub Logo Prompt

Use this prompt in Chat or an image generation tool:

```text
Design a simple but cool logo for a solo web studio called "Kuba's Web Dev Hub".

Create a clean horizontal wordmark with a small icon that can also work on its own as a favicon or social avatar. The style should feel modern, calm, sharp, and trustworthy, not playful or corporate-generic. Keep it minimal and memorable.

Requirements:
- Exact business name: Kuba's Web Dev Hub
- Include a small icon plus the full wordmark
- Transparent background
- Vector-style, crisp edges, no mockup presentation
- Strong readability at small sizes
- Simple enough for a website header, favicon, and social profile
- Use restrained colors inspired by deep ink navy and muted copper
- Optional palette: #182836, #32495d, #b05735
- Typography should feel refined and modern, not futuristic
- The icon should suggest web craft, structure, or a digital hub without using obvious globe, cursor, or code bracket cliches

Avoid:
- 3D effects
- mascots
- neon or cyberpunk styling
- overly busy symbols
- startup-style gradients everywhere
- cartoonish or gamer aesthetics
- generic AI-logo look

Output:
- 3 polished logo concepts
- each on a transparent background
- one should be the strongest recommended direction
```
